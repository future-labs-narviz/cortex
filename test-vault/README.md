# Test Vault

This vault serves as a test environment for the Cortex knowledge management system. It contains sample notes, patterns, and session captures used for development and testing.

## Structure

- **sessions/** - Session capture notes documenting development work
- **projects/** - Project-specific notes (Auth Module, Payment System, Cortex App)
- **patterns/** - Design patterns and architectural decisions
- **decisions/** - ADR-style decision records
- **voice-notes/** - Transcribed voice notes
- **captured/** - Auto-captured notes from various sources
- **daily/** - Daily notes and logs
- **plans/** - Planning documents
- **.cortex/** - Cortex configuration and note templates
